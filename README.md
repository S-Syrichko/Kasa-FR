# Kasa-FR

## Backend

This project is a frontend website for apartment rentals. <br>
The project was made from this <a  href = "https://www.figma.com/file/qEno0LwL4ZLkWyeY59kxp1/UI-Design-Kasa-FR"  target="_blank">Figma model</a>
  

## Start project

### With npm

`npm install` then `npm run start` will launch the project.

#### Available commands

`npm run build` to build project<br>
`npm run predeploy` : runs npm build<br>
`npm run deploy` : runs a deploy script for gh-pages<br>
